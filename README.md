# seewo-yuanshen

利用 vbs 劫持快捷方式来提前替换图片

### 优点
- 不需要管理员权限（设置自启）
- 无需后台常驻

### 使用
执行`Setup.hta`

- 若要删除，删除伪快捷方式即可
